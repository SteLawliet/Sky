搜索安装插件小技巧

进入putty
1、先更新源
opkg update
2、按照关键字搜索软件包名称
替换命令中为你的关键字，命令已经忽略大小写
opkg list |awk ‘BEGIN{IGNORECASE=1}//{print $1}’
比如搜索s-s
opkg list |awk ‘BEGIN{IGNORECASE=1}/s-s/{print $1}’
3、你所熟知的opkg install .ipk了复制代码*
