Aware

实现了Aware接口的bean在初始化时可以注入Spring容器内部基本组件 如IOC容器，内置embeddedValue解析器

  • ApplicationContextAware 注入IOC容器
  • BeanNameAware 注入当前bean的name
  • EmbeddedValueResolverAware 注入对类似“${} ,#{}”的字符串解析
