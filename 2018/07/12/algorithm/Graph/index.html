图的序列化表示

  1. 临接表法

  2. 临接矩阵

a(i,j)=1时表示俩点相连

i\j 0 1 2 3
0 0 1 0 0
1 1 0 1 0
2 0 1 0 0
3 0 0 0 0

由于是无向图,矩阵是基于对角线对称的,所以数据有一半的可以省略,可以把矩阵优化为下三角的部分,从左到右一行一行的放入大小为(n*n+1)/2的一维数组中

原本a(i,j)的值为现在a[(i*i+1)/2+j]的值.(a为优化后的数组)

数组生成图

(java中为了更方便的计算图的相关算法预先记录了相关冗余的属性,所以建立对象时没有采用以上的优化的一维数组来表示图,不过可以在序列化采用下三角矩阵的数组来存储图的信息)

建立图的java对象:

  • HashMap<Integer,Node> NodeMap 所有结点的Map
  • HashSet<Edge> edgeSet 所有边的Set
  1. 结点 Node
  • Object value 结点的值
  • int in 表示有in个结点指向this
  • int out表示有out个结点指向其他结点
  • List<Node> nextList 本结点指向的其他结点的集合
  • List<Edge> edgeList 从本结点出发的边的集合
  1. 边 Edge
  • int weight 边的权重
  • Node from 这条边由form结点出发
  • Node to 这条边是指向 to结点

2维数组生成图的结构

前面的序列化表示关注点是图的每个顶点.下面二维数组的表示是基于每一条边的.

二维数组matrix中的每个一维数组为[权值,form,to]
(matrix简写为arr)
遍历matrix中的每个一维数组 0 -> arr.length-1

  • arr[i][0] => 权重
  • arr[i][1]=> int form
  • arr[i][2]=> int to
  • if graph.nodeMap中没有form -> put(form,new Node(from))
  • if graph.nodeMap中没有to -> put(form,new Node(to))

宽度优先遍历 BFS

以距离源结点的层数来遍历,距源结点近的先输出.

1.利用队列实现
2.从头结点开始依次按照相同广度添加到队列中.然后弹出,打印
3.每弹出一个节点,把该节点所有没有进过队列的临接点放入队列
4.直到队列变空

深度优先遍历 DFS

从头结点开始一直走下去.直到所有的路走完才依次退回上一个结点.退回如果有其他分支后也一直走下去.直到所有的路走完才依次退回上一个结点.(走过的结点不走)

  1. 利用栈实现
  2. 从源结点开始把结点按照深度放入栈,然后弹出
  3. 每弹出一个结点,把该结点其中一个没有进过栈的邻接点放入栈.并打印
  4. 直到栈变空

具体实现:
传入要遍历的图的头结点head

  1. Stack<Node> stack 准备一个存储Node的栈
  2. HashSet<Node> set 准备一个存储Node的Set用来判断是否遍历打印过
  3. 添加head到set和stack中,打印头结点的val
  4. .while - stack为空
    1. pop出stack顶 => Node cur
    2. for each cur的nextList =>Node next ,判断next是否在set中添加过
      1. true - continue;

        在set中添加过说明已经在前面被遍历打印过

      2. false :
      3. 先添加cur到栈中
      4. 然后添加next到栈中
      5. set中添加next进行标记.然后打印next的val
      6. beak;结束遍历

拓扑排序

具体实现

HashMap<Node,Integer> inMap 记录所有结点的入度

Queue<Node> zeroInQueue 记录所有0入度的结点

  1. 从zeroInQueue中弹出结点.添加到返回结果的list中
  2. 遍历弹出的结点的邻接点集合nextList
  3. 每个邻接点的入度-1,如果入度-1后为0则添加到zeroInQueue队列中.
  4. 重复1,直到zeroInQueue为空

zeroInQueue保证先进先出.所以后继-1 n次后才变为0的也会根据进入的顺序依次弹出

最小生成树

kruskal算法

  1. 先从最小全值的边开始连接.
  2. 放弃产生回路的边
    具体实现

prim算法

  1. 先任意选择一个node
  2. 把该node的所有边加入到优先级队列(小根堆)中.
  3. 一直从队列中取出最小边.
  4. 如果指向的结点也没有入过set中.把该结点的所有边加入到队列中
  5. 重复2直到队列为空.
