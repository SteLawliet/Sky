归并排序MergeSort

2018-06-13
  • algorithm
    • sort

时间复杂度O(nlogn),额外空间复杂度O(n);

  1. 分治,中间分开mid=(l+r)/2

  1. 递归调用结束条件if（l==r) return;

  2. 分别处理递归 l -> mid和mid+1 -> r

  1. 合并递归结束的 l ->mid和mid+1 -> r

    1. 创建辅助数组help[]，长度为r-l+1;

      1. 使用外排的方式把 l -> mid和mid+1 -> r中小的数放进help[]中;

      2. 把排序好的help[]复制进要排序的数组中l -> r;
