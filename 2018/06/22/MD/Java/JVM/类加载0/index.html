加载->验证->准备->解析->初始化->使用->卸载

加载

加载需要完成以下工作

  1. 根据类的全限定名来读取对应的class文件的二进制字节流
  2. 在方法区中根据字节流的静态结构生成相关的运行时数据结构
  3. 在堆中生成类的java.lang.class对象，通过这个对象访问方法区中的数据
