mysql语句过长PacketTooBigException

com.mysql.jdbc.PacketTooBigException: Packet for query is too large (3158064）

shell进入mysql

set global max_allowed_packet = 2*1024*1024*10｀

然后关闭连接

重新进入运行

show VARIABLES like '%max_allowed_packet%';

查看下max_allowed_packet是否编辑成功
