mac VMware config static ip

2018-09-05
  • linux
    • ip

修改VMWare的vmnet8配置文件/Library/Preferences/VMware\ Fusion/networking，指定获取IP为静态以及修改网段为自定义的网段，如下：

VNET_8_DHCP 修改no

VNET_8_HOSTONLY_SUBNET 修改自定义静态ip的网段

修改网关

➜ vim /Library/Preferences/VMware\ Fusion/vmnet8/nat.conf

➜ ifconfig

vmnet1为host-only仅和mac连接

vmnet8为nat模式可以连外网,接下来修改linux的ip和这里的网段一样.

vmware修改完毕

配置虚拟机Linux

➜ ~ vim /etc/sysconfig/network-scripts/ifcfg-ens33
